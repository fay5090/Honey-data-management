🍯 Honey Stock Tracker App

A simple and efficient app to track honey stock, sales, and bottle sizes for small businesses.

📌 Overview

This app helps manage honey inventory by allowing you to:

Track different storage batches
Record incoming stock (e.g., 3L containers)
Sell honey in flexible quantities (1L, 2L, 3L, 5L)
Automatically calculate remaining stock

Perfect for small-scale honey sellers who want a clear and organized way to manage their business.

✨ Features
📦 Separate storage boxes for different batches
➕ Add new stock (e.g., number of 3L containers)
🛒 Record sales in different litre amounts
💰 Fixed pricing (e.g., 1L = 1500 KES)
📊 Live stock updates after each sale
🔄 Easy to update and manage inventory
🛠️ Tech Stack
HTML
CSS
JavaScript

(Simple and lightweight — no backend required)

🚀 How to Use
Open the app in your browser
Add a storage box
Enter how many 3L bottles you received
Record sales based on litres sold
The app updates remaining stock automatically
💡 Example
Add: 10 bottles (3L each) → Total = 30L
Sell:
1L → 1500 KES
2L → 3000 KES
5L → 7500 KES

Stock updates instantly after each transaction.
